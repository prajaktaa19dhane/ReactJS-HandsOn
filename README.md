## Getting Started

Follow these steps to get TaskBlast up and running on your local machine:

1. Clone this repository to your local environment.
2. Install the necessary dependencies using `npm install`.
3. Start the development server with `npm start`.
4. Open the app in your web browser.

## Usage

- Enter a task in the input field and click "NEW" to add it to your to-do list.
- Click "Delete" next to a task to remove it from the list.
- Click a task to mark it as completed.
